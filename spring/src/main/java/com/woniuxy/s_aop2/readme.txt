AOP 9 大概念：

1. 核心业务关注点：   就是核心业务
2. 横切关注点： 就是非核心业务
3. 通知：首先通知是一个方法，只不过该方法上有特定的注解：
	@Before @After @AfterReturning @AfterThrowing @Around
	通知中，封装了3种信息： 在什么时候，什么地点，干什么事情！
4. 切点：
	用于控制在什么地点织入切面
5. 织入：
	spring ioc容器，根据通知所指定的3样信息（ 在什么时候，什么地点，干什么事情）
	把目标对象包装为代理对象的动作过程。
6. 目标对象
	被包装的对象
7. 代理对象
	将切面与目标对象织入到一起以后，得到的对象。
8. 连接点(JoinPoint  ProceedingJoinPoint)
	连接点就是能够被织入切面的地点的全集。
	但实际上给哪个地点加入通知，取决于切点表达式。
	在spring中，连接点就是方法。 
9. 切面：
	对多个通知的封装。
